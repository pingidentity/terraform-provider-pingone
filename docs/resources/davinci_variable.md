# pingone_davinci_variable (Resource)

Resource to create and manage a DaVinci variable.

## Example Usage

```terraform

  environment_id = var.environment_id

  context   = "company"
  data_type = "string"
  mutable   = true
  name      = "region"

  display_name = "Region Variable"

  value = {
    string = "northamerica"
```
